# Board Report Template — Formal Board-Level Report Structure

> Use this template for reports presented to a board, investors, or equivalent senior governance audience. For regular operational reports, use `templates/bi_report_template.md`.

## Board Report Writing Rules

1. **Lead with the most important fact.** Do not make the board read to paragraph 4 for material news.
2. **Every number has a comparison.** State the value AND the baseline (target, prior period, or threshold). A number without comparison is uninformative.
3. **Bad news first, not last.** Burying concerns erodes trust when they surface later.
4. **Decision required = explicit ask.** If a board decision is needed, state it directly in Section 7 — not buried in narrative.
5. **No vanity metrics.** Only include metrics that connect to decisions or material risk.
6. **One page if possible.** The appendix is for supporting data. The main report should fit one page of dense information.
