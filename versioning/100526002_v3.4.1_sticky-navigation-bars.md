## Cél

A workspace navigáció főmenü sávja (tabs layout mode-ban) és az almenü sávok (Naptár, Erőforrások) görgetéskor eltűntek. A fix visszaállítja a sticky viselkedést mindkét layout mode-ban (sidebar + tabs), a helyes stacking offset kiszámításával.

---

## Érintett fájlok

| Fájl | Változás |
|------|----------|
| `src/components/enterprise/WorkspaceDashboard.tsx` | CSS custom property-k az outer div-en; főmenü TabsList `sticky`; Naptár almenü TabsList `sticky` |
| `src/components/enterprise/resources/ResourcesTab.tsx` | Erőforrások almenü TabsList `sticky` |

## Implementációs részletek

### CSS custom property megközelítés

A sticky top-offset értékeket CSS változók tárolják a közös ancestor container `style` prop-ján:

```tsx
// WorkspaceDashboard.tsx — outer div
style={{
  '--ws-header-h': '53px',                                    // header: py-2 + h-9 + border
  '--ws-main-tabs-h': layout === 'sidebar' ? '0px' : '65px', // main tabs: py-2.5 + h-11 + border
} as any}
```

Az almenük CSS custom property-ken keresztül öröklik az értékeket — nem kell új propot felvenni `ResourcesTab`-ba vagy más alosztályba.

### Egyes elemek sticky osztályai

**Főmenü TabsList** (tabs mode):
```
sticky top-[var(--ws-header-h)] z-20
```

**Naptár almenü TabsList**:
```
sticky top-[calc(var(--ws-header-h)_+_var(--ws-main-tabs-h))] z-10 bg-background border-b rounded-none
```

**Erőforrások almenü TabsList**:
```
sticky top-[calc(var(--ws-header-h)_+_var(--ws-main-tabs-h))] z-10 bg-background border-b rounded-none w-full
```

### Tailwind arbitrary value szintaxis

Az aláhúzás (`_`) a Tailwind arbitrary értékekben szóköznek felel meg a generált CSS-ben:
`top-[calc(var(--ws-header-h)_+_var(--ws-main-tabs-h))]`  
→  
`top: calc(var(--ws-header-h) + var(--ws-main-tabs-h));`

### Layout mode viselkedés

| Mode | Főmenü sáv | Almenü sticky top |
|------|-----------|------------------|
| `tabs` | `sticky top-[53px] z-20` | `calc(53px + 65px) = 118px` |
| `sidebar` | `sr-only` (nem látható) | `calc(53px + 0px) = 53px` |

---

## Elfogadási kritériumok

- ✅ Tabs layout: görgetéskor a főmenü sáv a header alatt rögzített.
- ✅ Tabs + sidebar layout: a Naptár és Erőforrások almenü sávok görgetéskor láthatóak maradnak.
- ✅ Sidebar layout: főmenü TabsList sr-only marad, almenük csak a header offsetet veszik figyelembe.
- ✅ A sticky sávok `bg-background` háttérrel fedik a guruló tartalmat.
- ✅ Nincs regresszió a v3.4.0-ban szállított funkciókon.
