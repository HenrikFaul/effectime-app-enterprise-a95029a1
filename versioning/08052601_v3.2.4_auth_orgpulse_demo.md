## Demo workspace contents
Per call, the seeder creates inside a fresh workspace:

| Module | Records | Notes |
|---|---|---|
| Auth users | 3 | `demo-<wsid>-N@demo.effectime.local`, email_confirmed |
| Profiles | 3 | display name set |
| Memberships | 3 | 1 resourceAssistant + 2 members, with team / city / business_role / hours |
| Offices | 3 | Budapest / Debrecen / Szeged, members auto-linked |
| Teams | 4 | Engineering / Product / Design / Operations |
| Leave types | 4 | Éves / Beteg / Otthoni / Fizetés nélküli |
| Holidays | 3 | recurring HU national holidays |
| Skills | 7 | TS / React / Node / Postgres / Figma / PM / Scrum |
| Member skills | 9 | 3 per member with proficiency |
| Leave quotas | 3 | 25 + 5 carryover, vacation type |
| Leave requests | 18 | mix of approved / rejected / pending across past + future |

All inserts are workspace-scoped. Existing FK cascade rules on `enterprise_workspaces` (verified across 50+ child tables) guarantee deletion of all rows when the workspace is deleted, so no orphans.

## Org Pulse compact header button
- Trigger: `Activity` icon + label "Org Pulse" (label hidden on `xs`)
- Alert detection: counts cells from {missing_org_unit, missing_manager, missing_contract, missing_leadership, open_approvals_long} with non-zero, non-suppressed value.
- Badge: destructive Badge with the count, only when `alertCount > 0`
- Popover content: 360px, two-column grid of all eight pulse cells, alert cells get `border-destructive/40 bg-destructive/5` styling, manual "Frissítés" link, k≥5 privacy badge preserved.

## Auth button rules
- Header in `WorkspaceDashboard` and `Enterprise` pages was already correctly authenticated-only — no change needed there.
- `Landing` was the offender; now the secondary CTA renders only when `!user`, and the primary CTA routes to `/app` for signed-in users.

## How to invoke seeder manually
```bash
curl -X POST \
  -H "Authorization: Bearer <user_jwt>" \
  -H "content-type: application/json" \
  -d '{"name":"My Demo"}' \
  https://oezlzzmzzvbvinuysxaz.supabase.co/functions/v1/seed-demo-workspace
```

Returns `{ ok: true, workspace_id, members_created }`.
