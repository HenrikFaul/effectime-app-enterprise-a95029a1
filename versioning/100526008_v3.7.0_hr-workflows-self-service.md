## Üzleti szabályok megvalósítva

| Szabály | Hol |
|---------|-----|
| Template → instance → tasks lánc | `hr_workflow_create_instance` RPC automatikusan materializál |
| due_date = instance.due_date + offset | RPC-ben SQL számítás |
| Assignee task frissítés | `hr_workflow_update_task` — admin VAGY az assignee saját maga |
| Pirosnál: overdue task jelzés | `isPast(due_date)` → piros badge/szín |
| Sárgánál: soon-due (3 nap) | `isWithinInterval(…, addDays(now, 3))` |
| Admin folyamat-zárás | `hr_workflow_close_instance` — completed vagy cancelled |
| Önkiszolgáló portál: egységes view | `EmployeeDashboard` 4 kártya, meglévő táblákból aggregál |
| Bérelőkészítési ready-check | `PayrollReadinessPanel` — useMemo alapú, 0 extra lekérdezés |
