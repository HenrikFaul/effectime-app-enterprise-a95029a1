## Why this PR is small

A blanket conversion of 648 hardcoded color uses across 86 files in one PR is a high regression risk. The user has repeated multiple times: "ne legyen regresszió". This PR therefore ships the **foundation** that lets every subsequent batch land safely:

1. The missing semantic token (`--info`) — without it, every blue/sky-themed panel is uncovertable.
2. The audit document with the full inventory, replacement matrix, prioritized 8-batch roadmap, and accessibility-by-default contract.

Subsequent PRs (A1–A8) execute the file conversions in screenshot-QA'd batches.
