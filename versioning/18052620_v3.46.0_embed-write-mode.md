# v3.46.0 — Embed Write Mode

**Date:** 2026-05-18
**Branch:** claude/analyze-repo-structure-dE8F4
**Type:** Feature + Bugfix

## Summary

Write-enabled embed tokens: CRM operators can now assign and remove shift assignments directly from the embedded iframe — without an Effectime login. Includes a critical bugfix for `create_embed_token` which was failing due to `gen_random_bytes` not found in the restricted search path.

## Changes

### Frontend
- **`EmbedCapacityView.tsx`** — Write panel per cell (assigned list + available members with +/× buttons); shows assigned names in cells when `can_write`; calls `embed_assign_shift` / `embed_remove_shift` RPCs
- **`EmbedShiftRosterView.tsx`** — Write mode: × button on shift badges to remove, + button on empty cells to assign; amber `✏ write` badge in header; `membership_id` in Member interface; `can_write` in EmbedData interface
- **`EmbedManager.tsx`** — Create dialog: `can_write` checkbox with description; `can_write` column in token list with Write/Read-only badge; passes `_can_write` to RPC; `Pencil` icon import
- **`en.ts` / `hu.ts`** — Added: `col_write`, `can_write_field`, `can_write_desc`, `write_badge`, `read_only_badge`

### Backend (migration applied to Supabase project `oezlzzmzzvbvinuysxaz`)
- `enterprise_embed_tokens.can_write boolean NOT NULL DEFAULT false` — new column
- `create_embed_token` — fixed `extensions.gen_random_bytes(32)` + `_can_write` param
- `get_embed_view_data` — returns `can_write` + `membership_id` in members
- `embed_assign_shift(text, uuid, uuid, text, date, uuid)` — new anon RPC, validates token + can_write + office scope
- `embed_remove_shift(text, uuid)` — new anon RPC, validates token + can_write + workspace scope

### Migration file
`supabase/migrations/20260518160000_v3_46_0_embed_write_and_fix.sql`

## Non-regression notes
- Existing read-only tokens unaffected (`can_write DEFAULT false`)
- `embed_assign_shift` uses `ON CONFLICT (user_id, shift_date) DO UPDATE` — idempotent upsert
- No UI changes to existing authenticated views
