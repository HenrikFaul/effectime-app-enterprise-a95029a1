## Design Concept

### Pan / Zoom
- **Transform layer**: Inner content div uses `transform: translate(${offsetX}px, ${offsetY}px) scale(${scale})` with `transform-origin: 50% 0` (top-centre anchor).
- **Drag**: `onMouseDown` / `onMouseMove` / `onMouseUp` / `onMouseLeave` on the outer container. Deltas accumulated into `offsetX` / `offsetY` refs.
- **Click vs drag**: `hasDragged` ref + 6 px threshold (`DRAG_THRESHOLD`). Capture-phase `onClickCapture` swallows the synthetic click event after a drag — employee drawer does not open unintentionally.
- **Wheel zoom**: `onWheel` with `preventDefault()`, `ZOOM_STEP = 0.15` per notch, clamped `[MIN_SCALE=0.2, MAX_SCALE=2.5]`.
- **Transition**: `transition: 0.18s ease` for zoom button clicks; disabled (set to `'none'`) while `isDragging.current` is true to avoid lag during drag.

### Zoom Controls
```
[−] [100%] [+]  |  [↺]
```
- Positioned `absolute bottom-4 left-1/2 -translate-x-1/2 z-20`
- `onMouseDown` stop-propagation prevents accidental pan start from control clicks
- Percentage label doubles as a reset button (click → 100% / origin)
- Separator `div` (1px border) between zoom group and reset icon

### Fullscreen Popup
- Toolbar button: `Maximize2` icon + "Teljes nézet" label; visible only when `view === 'premium'`
- Dialog: `max-w-[95vw] w-[95vw]` with `DialogHeader` (title + Sparkles icon)
- Content: `<OrgChartPremiumView containerHeight="calc(90vh - 80px)" ...>` — reuses existing component, no duplication
- `onOpenChange` handles backdrop-click / Esc close automatically via Radix

## Extension Points

1. **Pinch-to-zoom (touch)**: Add `onTouchStart/Move/End` handlers for mobile support.
2. **Fit to screen**: Button that auto-calculates scale+offset to center all roots in the viewport.
3. **Mini-map**: Small overview panel in the corner showing current viewport position within the full diagram.
4. **Export PNG**: `html2canvas` of the transform layer for a shareable org chart image.
