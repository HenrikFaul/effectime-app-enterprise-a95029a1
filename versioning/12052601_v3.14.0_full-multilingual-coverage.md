## Translation methodology

- **English** as canonical source; all source strings reviewed for clarity before translation
- **Hungarian** used as translation reference/memory for CS/SK/PL where helpful
- **Czech**: enterprise HR terminology, professional Czech loanword conventions for "workflow", "onboarding", "Sprint" retained where standard in Czech tech
- **Slovak**: closely mirrors Czech but with Slovak-specific morphology (kľúč, oprávnenia, udalosti)
- **Polish**: Polish professional HR/IT terminology; "Workflow" → "Przepływ pracy", "Allowance" → preserved as "Allowance" in equivalent PL context (already was "Keret" in HU, CS uses "Kvóta")
