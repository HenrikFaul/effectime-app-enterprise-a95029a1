## Összefoglaló
Enterprise B2B modul 1. fázis: Workspace + Membership + Roles alapok.

### Implementált funkciók
1. **Enterprise DB schema** (SYN-168):
   - `enterprise_workspaces` tábla (név, leírás, timezone, locale, beállítások)
   - `enterprise_memberships` tábla (user-workspace kapcsolat, role, status, team, location)
   - `enterprise_invitations` tábla (email-alapú meghívók, token, lejárat)
   - `enterprise_role` enum: owner, resourceAssistant, member
   - `enterprise_membership_status` enum: active, invited, suspended, removed

2. **Permission matrix** (SYN-170):
   - `has_enterprise_role()` SECURITY DEFINER funkció — role-alapú ellenőrzés
   - `is_enterprise_member()` SECURITY DEFINER funkció — aktív tagság ellenőrzés
   - RLS policies: workspace CRUD, membership CRUD, invitation CRUD — mind role-alapú

3. **Enterprise UI** (SYN-169):
   - `/enterprise` route — workspace lista és dashboard
   - Workspace létrehozás dialog
   - Workspace dashboard (tabbed: Tagok, Meghívók, Beállítások)
   - Member lista role-váltással, felfüggesztéssel, eltávolítással
   - Meghívó küldés dialog (email + role)
   - Meghívó lista törlési lehetőséggel
   - Workspace beállítások szerkesztése (owner only)
   - ProfileMenu-ben Enterprise link
