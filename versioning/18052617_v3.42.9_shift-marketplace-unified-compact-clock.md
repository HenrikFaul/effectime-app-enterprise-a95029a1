# v3.42.9 — Unified Shift Marketplace tab, Past-shift filter, Compact Clock-in layout

**Date:** 2026-05-18
**Branch:** claude/analyze-repo-structure-dE8F4

## Summary

Three UX consolidation improvements to the Employee Self-Service dashboard.

## Changes

### 1. Past open shifts filtered out (OpenShiftPanel)
- Added `const todayStr = format(new Date(), 'yyyy-MM-dd')` guard in the `visible` filter
- Any open-shift request with `shift_date < todayStr` is excluded before role/skill matching
- Rationale: past shifts are not actionable; showing them creates confusion

### 2. "Nyitott műszakok" merged into Shift Marketplace card (ShiftMarketplacePanel + EmployeeDashboard)
- `ShiftMarketplacePanel` now has three tabs: `'shifts' | 'open' | 'mine'` (default: `'shifts'`)
- First tab renders `<OpenShiftPanel workspaceId membershipId noCard />` — no Card nesting
- Standalone `<OpenShiftPanel>` block and its import removed from `EmployeeDashboard`
- Added `ClipboardList` icon import to `ShiftMarketplacePanel`

### 3. Compact side-by-side layout (EmployeeDashboard + ClockInPanel)
- Attendance stats Card + ClockInPanel wrapped in `grid-cols-1 md:grid-cols-2 gap-4 items-start`
- `ClockInPanel` receives `compact` prop:
  - Hides large `text-4xl` live clock (full date/time display)
  - Shows small `text-sm` time in the card header instead
  - Hides the "Today's timeline" event list at the bottom
- Attendance stats grid: `grid-cols-2` (was `sm:grid-cols-4`) to fit half-width column
