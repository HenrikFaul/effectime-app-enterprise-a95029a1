## Edit-mode gate

| Állapot | Pencil gomb | Save gomb | Batch + Készenlét gomb | Cellák kattinthatók? |
|---------|-------------|-----------|------------------------|----------------------|
| Server read-only (submitted+) | ❌ | ❌ | ❌ | Nem |
| Editable, edit-mode OFF | ✅ Szerkesztés | ❌ | ❌ | Csak megtekintésre |
| Editable, edit-mode ON | ❌ | ✅ Módosítások mentése | ✅ | Igen, drag-szel is |

A sárga „Szerkesztésre megnyitva" badge és helper banner jelzi az aktív szerkesztést. Hónap-váltáskor és status-flippelésnél automatikusan kilép edit-mode-ból.

## Batch kitöltés form

- Kezdő dátum, befejező dátum (a hónap határain belül kötve)
- Munkakezdés / munkavégzés időpont (`type="time"`)
- Típus: Normál munkaidő / Túlóra
- Toggle-ek: Hétvégék kihagyása · Éjszakai munka auto-jelölés · Meglévő felülírás
- Live count: „N nap lesz kitöltve" + figyelmeztetés a már létező rögzítésekre
- Apply → loop `upsertSegment` napra, eredmény: 1 toast (ok / kihagyott / hiba)

## Drag-select pattern (újrahasznosítható)

- Pointer events: `onPointerDown` cellára, `onPointerMove` gridre, `onPointerUp` cellára
- `data-day-cell data-date="YYYY-MM-DD"` minden cellán
- Touch: `document.elementFromPoint().closest('[data-day-cell]')`
- `useRef` az állapotnak — nincs re-render minden pointer-move-on
- `touch-action: none` a gridre edit-mode alatt (mobil scroll elkerülése)
- Vizuális preview: ring-2 + amber bg a hover-elt cellákon
- Globális `pointercancel` cleanup
- Single click (moved=false) → napi szerkesztő; drag (moved=true) → batch dialog

## Smoke test forgatókönyv

1. Időnyilvántartás → Saját idő → status: `draft`
2. Cella kattintható-e? **Nem** — előbb „Szerkesztés" gombra kell kattintani
3. „Szerkesztés" → badge megjelenik, banner megjelenik
4. Egy napra kattint → DayEditorDialog nyílik
5. Mehgse-zárás → semmi nem változik
6. „Batch kitöltés" gomb → dialógus megnyílik, hónap teljes, hétvégék kihagyva
7. Apply → toast „X nap kitöltve", napi cellák zöldek
8. Egér-húzás: nyomd le egér gombot egy napon, húzd át 3 napra → amber ring jelenik meg
9. Engedd el → Batch kitöltés dialógus megnyílik, kezdő/végdátum pre-fill-ve
10. Mobil érintés: ugyanaz a húzás touch-szal — működik
11. „Módosítások mentése" → badge eltűnik, cellák ismét nem-kattinthatók
12. „Benyújtás" gomb → status `submitted` → minden szerkesztő letiltva
