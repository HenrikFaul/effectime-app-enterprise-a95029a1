## Summary

Systematic layout audit against the `UI_UX_REFACTOR_UNIVERSAL_PROMPT.md`. Eliminates horizontal overflow on all breakpoints, adds missing grid stepping, and wraps potentially overflowing tables.

## Changes

### Phase 1 — Global CSS (`src/index.css`)
- `html { overflow-x: hidden }` — backstop for any escaped child overflow
- `body { overflow-x: hidden; max-width: 100%; scrollbar-gutter: stable }` — prevents horizontal scroll and CLS on scrollbar appearance
- `img, video { max-width: 100%; height: auto }` — ensures media never causes horizontal overflow

### Phase 1 — AppShell (`src/components/shell/AppShell.tsx`)
- Root flex container: added `overflow-x-hidden`
