# v3.49.3 — Landing page HU copy lectorate

**Date:** 2026-06-03
**Lens:** Senior UX Writer + Marketing/SEO manager + HU proofreader.

## Why
A felhasználó visszajelzése: a landing oldal magyar szövegei „borzalmas és magyartalan" hangzásúak voltak. Gombfeliratokat nem módosítunk; csak a body copy.

## Főbb javítások

### 1. T/V egységesítés — tegezés
- `cta_title`: „Próbálja ki már ma" → „Próbáld ki még ma"
- `cta_desc`: „Regisztráljon ingyenesen, hozza létre…" → „Regisztrálj ingyen, hozd létre…"
- (A gombok és nav linkek érintetlenek.)

### 2. Anglicizmusok kigyomlálása
- `showcase_title`: „people ops" → „napi HR-működéshez"
- `s1_desc` / `s1_bullet_1`: „accrual" → „felhalmozási"
- `problem_1_desc`: „post-itek" → „cetlik"
- `trust_3`: „Row-level security" → „Sor szintű biztonság"
- `vs_row_5_new` / `s1_desc`: „sérthetetlen audit log" → „változtathatatlan audit napló"
- `trust_title`: „Alapból enterprise szintű" → „Vállalati szint alapból"
- `trust_1`: „EU adat-rezidencia" → „EU-s adattárolás"
- `b3`: „iCal / naptár integráció" → „iCal- és naptárintegráció" (helyesírás)
- `b5`: „Email értesítések" → „E-mail értesítések"

### 3. Nyelvtan és ragozás
- `how_1_title`: „Hozd létre a munkaterületed" → „…munkaterületedet" (tárgyrag)
- `trust_desc`: „amit … elvár" → „amiket … elvár" (számbeli egyeztetés)
- `testimonial_1_quote`: „70%-kal csökkent a szabadság-adminisztráció" → „…szabadságok adminisztrációja"
- `how_2_desc`: „Tömeges import vagy meghívók. SSO Enterprise csomagon." → „Tömeges importtal vagy meghívókkal. SSO az Enterprise csomagban."
- `how_desc`: „Az adataid bármikor exportálhatók." → „Az adataidat bármikor exportálhatod." (alany/tárgy konzisztencia)

### 4. Konverziós / SEO csiszolás
- `hero_subtitle`: rövidebb, kulcsszó elöl („Szabadságkezelés, kapacitástervezés és jóváhagyások egy helyen…").
- `trusted_by`: „bíznak benne" antecedens-hiba → „14 ország operatív csapatai bíznak bennünk".
- `stat_*_label`: mondatkezdő nagybetűk eltávolítása (a számérték alatti címkék kisbetűsek, mint a többi SaaS landingen).
- `stat_4_label`: „Ország, ahol használják" → „ország használja".
- `problem_2_title`: „Lefedettségi hiányok túl későn" → „Késve derülnek ki a lefedettségi hiányok" (teljes mondat).
- `problem_3_title`: „Több telephely = káosz" → „Több telephely, sokszoros káosz" (kerüljük az egyenlőségjelet copyban).
- `s3_title`: „Két héttel előre lásd a hiányokat" → „Lásd a hiányokat két héttel korábban" (természetesebb szórend).
- `s4_title`: „Jóváhagyások a csapatod sebességén" → „Jóváhagyás a csapatod tempójában".
- `s4_bullet_1`: „1-kattintásos jóváhagyás" → „Egykattintásos jóváhagyás" (helyesírás).
- `s4_mock_sla`: „átl. 2ó 14p" → „átl. 2 ó 14 p" (mértékegység-szóköz).
- `vs_row_3_new`: „1-kattintás e-mailből · SLA-követés" → „Egy kattintás e-mailből · SLA-követéssel".
- `testimonial_3_quote`: „…ez a legnagyobb dicséret szoftverre." → „…ez a legnagyobb dicséret, amit egy szoftvernek adhatunk."
- `faq_title`: „Minden, amit kérdezhetsz" → „Mindenre van válaszunk".
- `faq_5_a`: „havidíjas, … számlázással. 50 fő felett mennyiségi kedvezmény." → „havidíj, … elszámolással. 50 fő felett mennyiségi kedvezmény jár."

- Interpolációk (`{{year}}`) megmaradtak.
