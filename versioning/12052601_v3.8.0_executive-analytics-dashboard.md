# v3.8.0 — Real-time Executive Intelligence Dashboard with Predictive Analytics

**Date:** 2026-05-12
**Branch:** claude/create-software-documentation-O7kj1
**Type:** New feature

## Scope

Added a dedicated `Analytics` tab (admin-only) to the workspace top navigation with a full executive intelligence dashboard.

## Components delivered

### `src/components/enterprise/analytics/AnalyticsDashboard.tsx`
- Props: `{ workspaceId: string; isAdmin: boolean }`
- 4 KPI summary cards (Scheduled Hours MTD, Labor Cost MTD, Absence Rate, Coverage Score)
- 6-month labor cost forecast AreaChart using Recharts
- 13-week coverage risk heatmap (Mon–Sun grid, color-coded: safe/low/medium/high/critical)
- Absence risk table sorted by combined sick + overtime risk score, red-highlighted critical rows
- All data fetched client-side from existing Supabase tables (no new edge functions)

## DB changes

No new tables for this feature. Reads: `enterprise_memberships`, `enterprise_member_rates`, `leave_requests`, `enterprise_attendance_periods`, `enterprise_coverage_rules`, `profiles`.

Migration file: `supabase/migrations/20260512210000_analytics_wellbeing_api_platform.sql` (shared with v3.9.0 and v3.10.0).
