## Implementált funkciók

### Phase 4: Approval Chain + Escalation (SYN-146)
- `ApprovalChainManager`: többlépcsős jóváhagyási lánc, lépés hozzáadás/törlés, szerepkör választás
- `enterprise_escalation_rules`: eszkaláció beállítások (óra küszöb, célszerepkör, owner értesítés)

### Phase 5: Audit Log
- `AuditLog`: szűrhető, kereshető audit esemény nézegető (művelet típus, keresés)
- `enterprise_audit_events` tábla immutable insert-only policy-val

### Phase 6: Enterprise Notifications
- `EnterpriseNotifications`: olvasott/olvasatlan kezelés, törlés, "mind olvasott" gomb
- Emoji ikonok eseménytípus szerint

### Phase 7: Export
- `ExportCenter`: CSV export dátumtartomány és státusz szűrővel
- BOM-os UTF-8 CSV, ünnepnapok integrálva, audit bejegyzés létrehozás

### Phase 8: Reporting Dashboard
- `ReportingDashboard`: KPI kártyák (összes, jóváhagyott, függőben, jóváhagyási arány)
- Recharts: státusz pie chart, típus bar chart, napi távollévők bar chart
- Időszak szűrő (7/30/90 nap)

### Phase 9: Rule Templates
- `RuleTemplateLibrary`: sablon CRUD, duplikálás, archiválás, JSON szerkesztő
