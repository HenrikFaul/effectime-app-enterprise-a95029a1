## Architecture decisions
- **Fat SECURITY DEFINER RPC** (`get_embed_capacity_planner_data`) callable by `anon` role — validates token internally, returns JSONB with offices, rules, shifts, holidays. No RLS changes.
- **Token stored as `encode(gen_random_bytes(32), 'hex')`** — 256 bits of entropy, not guessable.
- **Revocation is instant** — `revoke_embed_token` sets `is_active = false`; next RPC call returns error.
- **HashRouter path** — embed URL is `<origin>/#/embed/capacity_planner?token=<token>`.
