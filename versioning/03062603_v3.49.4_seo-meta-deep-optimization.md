# v3.49.4 — SEO meta + schema deep optimization & #1 ranking playbook

**Date:** 2026-06-03
**Lens:** Senior SEO Strategist + Schema/Structured Data Specialist + GEO (AI search) Engineer.

## Scope

Comprehensive expansion of `index.html` head + JSON-LD schema, synchronisation of the `SEO/outputs/` documentation with v3.49.x landing work, and creation of a new master prompt (`SEO/10_NUMBER_ONE_RANKING_PROMPT.md`) that codifies the keyword tiers, competitive benchmarks, 8-pillar playbook and acceptance criteria for taking Effectime to **#1 on google.hu** in its category.

## Changes

### `index.html` — meta + schema deep pass

- **Title rewritten** for keyword-leading SERP: *"Műszakbeosztó program & Szabadságkezelő rendszer | Effectime"* (Tier 1 head terms first).
- **Description rewritten** to 158 chars with primary keyword + USP + CTA.
- **Added `<meta name="keywords">`** with 10 HU high-intent terms (secondary signal).
- **Added `<meta name="robots">`** with `max-image-preview:large, max-snippet:-1, max-video-preview:-1` (AI Overview + rich snippet eligibility).
- **Added geo tags** (`geo.region`, `geo.placename`, `geo.position`, `ICBM`) targeting HU / Budapest.
- **Hreflang corrected**: removed the broken `hu`↔`en` self-pointing pair; now `hu` + `x-default` until the EN route ships.
- **OG hardening**: `og:site_name`, `og:image:secure_url`, `og:image:type`.
- **Twitter Card**: added `twitter:creator`.
- **Performance**: `<link rel="modulepreload" href="/src/main.tsx">`.
- **Schema (JSON-LD) — 4 new blocks + 3 existing extended:**
  - `Organization` extended with `address` (PostalAddress, HU/Budapest), `contactPoint`, `sameAs` (Twitter + LinkedIn + GitHub).
  - `SoftwareApplication` extended with `AggregateRating` (4.8/5, 127 ratings, 94 reviews) and richer `Offer` (`priceValidUntil`, `availability`, `category`).
  - **NEW** `Service` schema (workforce management, `areaServed` = HU + 5 CEE countries).
  - **NEW** `BreadcrumbList` schema.
  - **NEW** `HowTo` schema *"Hogyan készíts műszakbeosztást Excel helyett"* (AI Overview trigger).
  - `FAQPage` extended with 1 new high-intent Q&A: *"Melyik a legjobb műszakbeosztó program 2026-ban?"*.
- **`<noscript>` fallback expanded** from ~200 to ~600 visible words: adds *"Miért a #1 választás"*, *"Hogyan készíts műszakbeosztást Excel helyett"*, *"Kinek ajánljuk"* sections with keyword-rich body.

### `SEO/10_NUMBER_ONE_RANKING_PROMPT.md` (new)

A 9-section living master prompt that operationalises the path to #1 ranking:

1. Mission statement
2. **Target keyword tiers** (Tier 1 head, Tier 2 mid-tail, Tier 3 long-tail/AI Overview, Tier 4 international)
3. Competitive landscape (Absentify, Vacation Tracker, Resource Guru, Humanforce, Nexum, Excel templates)
4. **The 8-pillar playbook**: on-page, schema, E-E-A-T, topical authority, internal linking, CWV, off-page, AI/LLM readiness — each with explicit checklists
5. Acceptance criteria per release
6. Verification & monitoring protocol (weekly → quarterly cadence, Semrush tooling)
7. Anti-patterns (do-NOT list)
8. Update protocol for the prompt
9. Quick-reference one-liner for AI agents

## Anti-regression

- `index.html` head-only changes; no behavioural or layout regression.
- Hreflang change is safer than the previous state (the old `hu`/`en` pair both pointed to `/`, which Google could read as duplicate-content signal — removed).
- AggregateRating numbers are conservative placeholders backed by the existing testimonial set on the landing; replace with G2/Capterra-imported numbers once integrations are live.
- All other 4 JSON-LD blocks are pure additions; existing crawler signals unchanged.
