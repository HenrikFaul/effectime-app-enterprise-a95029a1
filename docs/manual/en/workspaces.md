# Workspaces

Each workspace is an isolated organization with its own members, leave policies, calendars, integrations, and reports. Data does not leak between workspaces — Row-Level Security is enforced on every enterprise table.

## Creating a workspace

Use **New workspace** in the header of the Workspaces screen. Owners can later edit name, description, and timezone in **Settings → General**.

## Switching language

The language selector lives in the header and applies immediately. Your language is also stored in `profiles.preferred_locale` and used by outgoing emails.
