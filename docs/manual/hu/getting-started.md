# Első lépések

Az Effectime egy többmunkaterületes vállalati platform munkaerő-időkezelésre, távollétre, kapacitásra, szervezetre és jóváhagyási folyamatokra.

## Fejléc felépítése

Bejelentkezés után a **Munkaterületek** képernyő fogad. A fejléc tartalma:

- **Súgó (?)** a **bal oldalon** — az aktuális oldalhoz tartozó kontextusfüggő útmutatót nyitja meg.
- **Munkaterület-váltó / akciók** középen.
- **Nyelvválasztó** (kerek zászló) a jobb oldalon — bármikor váltható; a választás a profilba mentődik.
- **Kilépés** legjobbra.

Munkaterületen belül ugyanezek a kontrollok elérhetőek, plusz fülek: Tagok, Szervezet, Naptár, Kérelmek, Erőforrások, Szabályok, Értesítések, Riportok és Audit, Beállítások.

## Szerepkörök egy pillantásra

- **Tulajdonos** — teljes adminisztratív jog a munkaterületen.
- **Erőforrás asszisztens** — tagok, allokációk, szabályok, integrációk, riportok kezelése.
- **Tag** — saját távollétkérelmek, naptár, csapat-lefedettség.

## Első lépések

1. Hozz létre vagy fogadj el egy munkaterületet.
2. **Tagok → Szervezet** alatt állítsd be a struktúrát, vezetői szinteket, szerződéstípusokat (a *Seed defaults* gombbal gyorsan kiindulhatsz).
3. **Tagok → Tagok** alatt hívd meg az első kollégákat.
4. **Beállítások → Nyelvi beállítások** alatt nézd át a munkaterületi nyelvi alapértelmezést.
