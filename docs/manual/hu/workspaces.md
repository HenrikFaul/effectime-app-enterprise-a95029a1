# Munkaterületek

Minden munkaterület egy izolált szervezet a saját tagjaival, szabályaival, naptárával, integrációival és riportjaival. Az adatok nem szivárognak munkaterületek között — minden enterprise táblán Row-Level Security érvényes.

## Új munkaterület

A Munkaterületek képernyő fejlécében az **Új munkaterület** gombot használd. A tulajdonos később a **Beállítások → Általános** alatt módosíthatja a nevet, leírást és időzónát.

## Nyelvváltás

A nyelvválasztó a fejlécben azonnal érvényesül. A választott nyelv a `profiles.preferred_locale` mezőbe kerül, és a kimenő emailek is ezt használják.
