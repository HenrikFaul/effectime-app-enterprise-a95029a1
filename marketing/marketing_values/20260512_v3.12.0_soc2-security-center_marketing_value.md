# Marketing Value — v3.12.0 SOC 2 / ISO 27001 Security Center

## Feature name and version
SOC 2 Type II / ISO 27001 Technical Controls & Security Center — v3.12.0

## One-sentence user-language description
Effectime now includes a built-in Security Center giving enterprise admins an immutable audit log, configurable data retention policies, GDPR data subject tools, and session management — the technical controls required for SOC 2 and ISO 27001 certification.

## Problem it replaces
Enterprise buyers can't approve a tool without a security audit trail, data retention controls, and GDPR compliance features. Without these, Effectime fails procurement review at 500+ employee companies and any regulated industry, public sector, or Western European organization — regardless of how good the scheduling features are.

## Primary beneficiary personas
- **IT Manager / CISO** — audit log with IP address, user agent, and state diff satisfies SOC 2 CC7.2
- **Data Protection Officer** — GDPR Article 17/20 tooling built in, no custom dev required
- **HR Director (enterprise procurement)** — security certificate becomes a sales conversation, not a blocker
- **Owner / CTO** — path to SOC 2 Type II audit is now realistic without a full engineering overhaul
