## User-language problem solved

Non-Hungarian managers using the workforce planner saw Hungarian labels ("Heti", "Havi", "Beosztva", "Extra (slot felett)") embedded in the English/German/Polish/Romanian/Czech/Slovak interface — creating a confusing, unprofessional experience. Additionally, when managers clicked the "broadcast" badge in the compact planner view, existing open-shift broadcasts disappeared with no way to cancel them, forcing a page reload to recover.

---

## Personas who benefit

| Persona | Benefit |
|---------|---------|
| Non-Hungarian managers (DE, AT, CS, SK, PL, RO, EN) | All planner UI labels now appear in their configured language |
| All managers using coverage planner | Compact-mode open-shift badge now shows all existing broadcasts with cancel buttons |
| Developers / integrators | TypeScript type contract corrected; no silent runtime type mismatches |
