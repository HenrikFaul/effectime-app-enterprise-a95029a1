# Marketing Value — v3.14.0 Full Multilingual Coverage

## Feature name and version
Full multilingual localization — Czech, Slovak, Polish, Hungarian — v3.14.0

## One-sentence user-language description
Every screen, button, message, and help text in Effectime now speaks your language — Czech, Slovak, Polish, or Hungarian — with no English fragments left behind.

## Problem it replaces
Users in Czech, Slovak, and Polish markets encountered English-language screens in the Developer API platform, empty help drawers, and inconsistent Hungarian labels (e.g. "Allowance", "Remaining", "Auto-create"). This created a perception of an unfinished product and reduced trust in enterprise sales conversations with non-English HR teams.

## Primary beneficiary personas
- **HR Manager (CS/SK/PL/HU)** — no longer sees English strings mid-workflow
- **IT Admin / Developer (CS/SK/PL)** — Developer API platform is now fully localized
- **Workspace Owner (all locales)** — Help drawer now provides context-aware guidance in their language
- **Enterprise Sales team** — Can demo the product in Czech, Slovak, or Polish without English artifacts

## Marketing-ready claim
"Effectime is fully localized in 5 languages — English, Hungarian, Czech, Slovak, and Polish — with zero English-only screens, giving HR teams in Central Europe a native-language enterprise experience."

## Claim types enabled
- **Trust / polish** — no English fragments in non-English UI
- **Market readiness** — product is now enterprise-ready for CZ/SK/PL markets
- **Compliance / professionalism** — HR and payroll software must feel native-language to enterprise buyers
- **Competitive differentiation** — most HR SaaS tools have partial or machine-translated CZ/SK/PL
