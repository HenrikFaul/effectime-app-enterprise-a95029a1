# Marketing Value — v3.41.3: Free-text search in position picker

**Version:** v3.41.3  
**Date:** 2026-05-17  
**Feature slug:** position-picker-search

## Problem in user language

When setting up a new position or advertising an open shift, managers had to click through a multi-level category tree to find the right position — even if they already knew the name. With dozens of categories and hundreds of positions, this was slow and frustrating.

## Solution in user language

Now you can just start typing. A search bar at the top of the position picker instantly filters all positions across every category. Click the result and jump straight to the skills review — no more tree-clicking.

## Personas that benefit

- **HR managers / schedulers** creating open shifts — faster shift setup reduces scheduling bottlenecks
- **Admins** adding positions to members via the Resources menu — less friction, fewer errors
- **Enterprise clients with large role catalogs** — the tree was unusable at scale; search makes the catalog practical

## Marketing claims enabled

- "Instant search across 100+ predefined positions — find any role in seconds"
- "Autocomplete position picker: no more clicking through endless category trees"
- "Smart catalog search cuts shift setup time dramatically"
