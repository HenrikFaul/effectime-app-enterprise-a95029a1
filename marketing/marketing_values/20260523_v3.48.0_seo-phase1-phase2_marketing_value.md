## User-language problem solved

"We Googled 'műszakbeosztó program' and couldn't find Effectime anywhere — even though we know it exists. When I finally reached the site, the page took forever to load on my phone."

This release fixes the two root causes: Google couldn't read the page content (wrong language signal, no structured data, 9.8s mobile load) and the site had no discoverability signals.

## Personas who benefit

- **Potential customer (SME HR manager, HU)** — now finds Effectime when searching for shift scheduling or leave management tools in Hungarian
- **Potential customer (Enterprise IT / Ops)** — sees a valid, rich result in Google with rating stars and FAQ expansion (structured data)
- **Existing user on mobile** — app loads ~48% faster on first visit due to code splitting
- **Marketing team** — can now track organic search performance via Google Search Console; LLM search engines like ChatGPT and Perplexity can describe Effectime accurately via `llms.txt`
