# v3.31.0 — Candidate Scheduling / ATS bridge — Marketing Value

## One-sentence description
Combined workforce + recruiting platform — schedule interviews against
your real team's leave calendar, candidates self-book via a public URL,
and Effectime auto-generates a first-week onboarding schedule on hire.

## Personas
- **Recruiters / Talent acquisition** — replaces Calendly + ATS stitching with a single tool.
- **Hiring managers** — see eligibility-validated slots that respect their team's actual availability.
- **Candidates** — clean public self-booking page; one-click confirmation.

## Claim types enabled
- **TAM expansion** — from "workforce management" (current employees) to "total talent management" (candidates + employees). Per the strategy doc: $400M candidate-scheduling sub-market.
- **Bundle deal premium** — WFM + recruiting in one contract justifies 30-50% higher contract value.
- **Eligibility advantage** — competitor candidate-scheduling tools (Calendly Enterprise, GoodTime) don't know about leave / shifts; Effectime does.
