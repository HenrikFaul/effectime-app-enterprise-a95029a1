# v3.20.0 — GDPR / WTD Compliance Engine — Marketing Value

## One-sentence description

Real-time labor-law compliance dashboard with per-workspace
jurisdiction selection (EU WTD, Hungarian Mt., German ArbZG, Austrian
AVRAG) — so a workforce schedule never silently breaches 48-hour
weekly limits or daily rest requirements.

## Personas helped

- **HR / People Ops** — green/yellow/red status at a glance, period
  date pickers, on-demand check that writes an auditable violation
  log.
- **Workforce schedule manager** — sees which members are approaching
  their weekly maximum BEFORE publishing next month's schedule.
- **Labor-inspector readiness** — every check writes an immutable
  `compliance_violations` row with actor, period, actual vs. limit,
  jurisdiction. Audit trail is permanent.

## Claim types enabled

- **Regulated-industry unlock** — healthcare, finance, public sector
  procurement frequently require evidence of working-time compliance.
  This dashboard is that evidence.
- **DACH market entry** — German ArbZG and Austrian AVRAG support
  positions Effectime as the only mid-market WFM tool with built-in
  multi-jurisdiction compliance.
- **Pro-tier upsell** — Freemium workspaces see a LockedFeatureNotice;
  upgrading to Pro unlocks the entire compliance suite. Per-feature
  upsell story is clear and concrete.
