# Marketing Value — v3.42.4: Gated Availability Paint Mode + Notification Fix

## Feature name and version
Dual-mode employee edit system + Open shift notification fix — v3.42.4

## One-sentence user-language description
Employees now set their availability through a dedicated, intention-first paint mode — no more accidental changes — while managers finally see the right confirmation when notifying only selected candidates.

## Problem it replaces
1. Employees could accidentally change their availability by tapping a calendar day without realising they were in edit mode — data integrity issue.
2. Managers clicking "Notify selected" received a confusing "Sent to all members" confirmation, undermining trust in the targeted-notification workflow.

## Primary beneficiary persona(s)
- **Employee / Team Member** who marks their own availability — gains confidence that the calendar is read-only until explicitly editing
- **Operations Manager / HR Manager** who posts open shifts — gets clear confirmation of exactly who was notified

## Marketing-ready claim
*"Your schedule calendar is now read-only by default — availability changes require intent, not an accidental tap. And when you notify only the right people, Effectime tells you exactly who received it."*

## Claim types enabled
- **Reliability** — no accidental availability overwrites
- **Precision** — targeted notifications confirmed with exact count
- **Trust** — UI confirms the action taken, not a generic success message
- **Professionalism** — paint-mode UX on par with dedicated workforce tools (Syncfolk-style)

## Content angles
1. **Product update / release notes:** "Employee calendar is now read-only by default. To update availability, tap Edit → Availability and paint days — past dates are locked."
2. **LinkedIn / social:** "We separated 'schedule editing' from 'availability marking' — two modes, two intentions. Drag to paint availability across multiple days on any device."
3. **Support deflection:** Proactively addresses support tickets for "my availability changed and I didn't do it."

## Funnel stage
- **Retention** — reduces frustration from accidental edits and misleading confirmations
- **Expansion** — paint-mode drag UX encourages employees to actively mark availability, enabling better shift coverage
