# v3.15.2 — Routing seed — Marketing Value

## Feature name and version

Routing seed (v3.15.2) — all 135 features now have a route_path and
Hungarian menu breadcrumb in the feature catalog.

## One-sentence user-language description

Platform operators can now see exactly where in the app each of the 135
features lives — as a real navigation tree, not a flat list — and can
audit / restructure the menu hierarchy directly from the Superadmin UI.

## Problem it replaces

Before this release, the FeatureTiersTab routing audit banner reported
"Hiányzó menu_path 135" — every feature was missing route + menu data, so
the routing tree (one of the central feature-management views) was
empty/broken and the audit checks were red across the board. The
underlying DB columns existed (added in v3.13) but were never seeded.

## Primary beneficiary personas

- **Platform owner / operator** — has a working navigation tree to see
  the full app surface at a glance.
- **UX designer / IA reviewer** — can review and propose menu
  restructures using the FeatureTiersTab routing audit banner +
  drag-and-drop reordering.
- **Sales / Customer success** — can hand a customer a route-by-route
  feature inventory generated from `docs/tiering/features.csv`.

## Marketing-ready claim

"Every Effectime feature mapped to its place in the app. 135 features,
135 routes, 135 menu paths — all visible in a single Superadmin view
that turns the catalog into an explorable navigation tree."

## Claim types enabled

- **Visibility** — the navigation tree is now a real tree, not a flat
  list with empty cells.
- **Compliance / audit** — restructures are tracked in
  `platform_audit_events` (v3.15.0 audit log), giving operators a clean
  record of every menu / route change.
- **Onboarding** — Sales can demo "here's our entire feature map" by
  scrolling the routing tree.

## Funnel stage

  Feature & Tier Management view to customers.
- **Trust** — the audit banner being green instead of red is a small
  but visible signal that the platform is internally consistent.
