# Marketing Value — v3.46.0 Embed Write Mode

**Version:** 3.46.0
**Date:** 2026-05-18
**Feature slug:** embed-write-mode

---

## Problem solved (user language)

CRM users who have the Effectime capacity planner or shift roster embedded in their portal couldn't actually do anything with it — they could only look. If a manager spotted a gap, they had to switch to a separate Effectime tab, log in, navigate, and fix it there. Context-switching is friction, and friction means gaps go unfilled.

---

## What's new

Embed tokens can now be **write-enabled**. A CRM operator sees the live roster or capacity planner in their portal and can directly click to assign or remove a shift — all without ever opening Effectime, creating an account, or leaving their workflow.

---

## Personas who benefit

| Persona | Benefit |
|---|---|
| **CRM operator / shift planner** | Can manage the schedule from the tool they already live in |
| **Workspace admin (owner)** | Controls exactly which tokens grant write access; revoke anytime |
| **Enterprise IT / developer** | Simple toggle when generating the embed snippet; no extra API integration needed |
