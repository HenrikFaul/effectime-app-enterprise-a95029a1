# v3.15.1 — Tiering follow-ups — Marketing Value

## Feature name and version

Tiering follow-ups (v3.15.1): platform audit log viewer + full feature
catalog localization, completing the v3.15.0 rollout.

## One-sentence user-language description

Platform operators can now read the audit log of every plan / tier / feature
change in a searchable, paginated UI, and Hungarian users see all 135
features in Hungarian (instead of English mixed with Hungarian as before).

## Problem it replaces

After v3.15.0 the audit table existed and the toggles wrote to it, but the
only way to read the log was a direct SQL query — useless for compliance
auditors and operators. Separately, Hungarian users opening the Superadmin
Feature & Tier view saw an "English chrome / Hungarian feature names"
mixed-language UI, which felt half-finished and undermined the localization
story.

## Primary beneficiary personas

- **Platform owner / SaaS operator** — has a real compliance-grade audit
  log they can hand to a SOC 2 / ISO 27001 auditor.
- **Customer success / Support engineer** — can answer "who turned this
  feature off?" without filing an engineering ticket.
- **Hungarian-language buyer** — every feature label is in Hungarian (135
  hand-localized strings) — sales demos no longer expose internal English
  strings.
- **End user / Admin in any locale** — tier names ("Freemium" / "Pro" /
  "Enterprise") and addon names ("Payroll Integration" / "AI Assistant")
  render in their language.

## Marketing-ready claim

"Audit log every plan change. See every tier toggle, every feature edit,
every routing change — with actor, before, and after — in an immutable
log that satisfies SOC 2 § CC7.2. Plus the entire 135-feature catalog
now in Hungarian and English, with three more EU languages on the way."

## Claim types enabled

- **Compliance** — concrete audit-log viewer screenshot for sales decks
  targeting SOC 2 / ISO 27001 buyers.
- **Localization** — "fully Hungarian-localized control plane" claim is
  now literally true, not aspirational.
- **Visibility** — paginated audit viewer with filters means operators
  can answer billing-related "who upgraded this tenant?" questions in
  seconds.
- **Trust** — immutable storage (no UPDATE / DELETE policy) is a hard
  guarantee, not a promise.
