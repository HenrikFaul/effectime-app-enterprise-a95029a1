# v3.30.0 — Plugin Marketplace — Marketing Value

## One-sentence description
Effectime is now a platform, not just a product — third-party developers
can publish plugins, workspace owners install them with one click, and
the ecosystem extends Effectime into verticals (healthcare, retail,
construction) without our core team writing vertical-specific code.

## Personas
- **Third-party developers** — build vertical extensions, collect rev share.
- **Workspace owners** — install plugins that solve their specific workflow gaps.
- **Effectime platform team** — TAM expansion at near-zero marginal dev cost.

## Claim types enabled
- **Platform premium** — per Salesforce AppExchange model, ecosystem
  moats add 40-60% to platform-vs-product valuation multiples.
- **Vertical reach** — healthcare, retail, construction extensions
  ship without core team allocating engineers per vertical.
- **Developer-ecosystem network effect** — each plugin is also a
  discovery channel.
