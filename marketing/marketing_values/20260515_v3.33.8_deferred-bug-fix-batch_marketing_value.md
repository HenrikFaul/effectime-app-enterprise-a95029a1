# Marketing Value — v3.33.8: Deferred Bug-Fix Batch

## Feature name and version
Platform reliability and internationalization hardening — v3.33.8

## One-sentence user-language description
A targeted set of bug fixes makes Effectime more reliable for large organisations, eliminates lingering Hungarian-only error messages, and ensures the leave-approval workflow filters work correctly on first use.

## Problem it replaces
- Managers using team or role filters in the Approval Inbox saw incorrect results on first load (stale filter state).
- Superadmin platform dashboards silently under-counted auth users on installations with more than 1000 users.
- Microsoft 365 calendar integration could fail to register a new OAuth connection without surfacing any error.
- The Export panel heading was untranslatable (hardcoded English).
- Conflict engine log output contained Hungarian text, making debugging opaque for non-Hungarian engineering teams.

## Primary beneficiary persona(s)
- **HR / Ops Manager** — Approval Inbox team/role filters now work correctly on first open; no need to reset filters to get accurate results.
- **Platform Admin / IT** — Superadmin hub now reports accurate user counts for large tenants; MS365 OAuth failures are visible rather than silent.
- **Engineering / DevOps** — Conflict engine logs are now English, making production debugging accessible to international engineering teams.
- **All locales** — Export heading is now translatable.

## Marketing-ready claim
"Effectime's reliability fixes ensure that approval workflows, admin dashboards, and calendar integrations work correctly at any scale — with fully internationalised UI throughout."

## Claim types enabled
- Reliability / correctness (filter bug fixed, OAuth error surfaced)
- Enterprise scale (>1000 user pagination in admin hub)
- Internationalisation completeness (no remaining hardcoded strings in reviewed surface area)

## Content angles
1. **"Built for large teams"** — Admin hub now handles platforms of any size without silent data truncation.
2. **"Approval workflows that just work"** — Manager filters show the right people from the moment you open the inbox.
3. **"No hidden errors"** — MS365 integration failures now surface immediately so admins can act.

## Funnel stage
Retention / trust-building:
- Reduces friction for existing customers on large installations.
- Supports enterprise sales conversations around reliability and internationalisation completeness.
