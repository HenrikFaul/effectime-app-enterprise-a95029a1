# v3.24.0 — Public REST API gateway + webhook dispatcher — Marketing Value

## One-sentence description

Effectime workspaces now have a public REST API gateway with rate-limited
Bearer-token auth + HMAC-signed webhooks — every Effectime workspace
can integrate into 6000+ tools via Zapier-class connectors and into
custom enterprise stacks via direct webhook subscriptions.

## Personas helped

- **CTOs / Heads of Engineering** at customer companies — finally a
  WFM tool whose data they can pipe into their own systems without a
  CSV export + cron job hack.
- **Effectime partner integrators** — Zapier/Make/n8n developers can
  build connectors against a documented, stable REST surface.
- **Internal IT teams** — webhook subscriptions let them react to
  events (leave.approved, schedule.published) in real time.
