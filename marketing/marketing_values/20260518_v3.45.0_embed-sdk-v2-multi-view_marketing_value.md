# Marketing Value: Embed SDK v2 — multi-view + snippet builder (v3.45.0)

## Problem solved
"We need to show different views in our CRM — sometimes capacity, sometimes who is working today." Now both views are embeddable with one token and a visual snippet builder — no developer needed.

## Personas who benefit
- **CRM admin at enterprise client** — picks view, office, height in a GUI and gets the exact `<iframe>` code in seconds.
- **Effectime workspace admin** — controls exactly which views each token can access.
- **Multi-site operations manager** — embeds a single-office filtered roster next to their CRM calendar.

- "Snippet Builder: configure and copy your iframe code in 30 seconds, no coding."
- "Filter by office — show only what's relevant to each team."
- "Monthly heatmap calendar: one glance shows your whole month's coverage."
