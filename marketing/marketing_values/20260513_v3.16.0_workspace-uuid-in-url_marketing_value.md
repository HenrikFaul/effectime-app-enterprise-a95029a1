# v3.16.0 — Workspace UUID in URL + Back-button fix — Marketing Value

## Feature name and version

Shareable workspace URLs + reliable browser navigation (v3.16.0).

## One-sentence user-language description

Every workspace now has its own URL — you can bookmark it, share it
with teammates, and the browser Back button always takes you to where
you were a moment ago, never randomly to the homepage.

## Problem it replaces

Two interlocking frustrations:

1. **No deep-link to a workspace.** Sales engineers, customer success
   reps, and operators couldn't paste a link to a specific customer's
   workspace into Slack or email — the link would just open the picker
   for whoever clicked it, and they'd have to find their way back.

2. **Back button dropped users to the landing page.** After picking a
   workspace and clicking through a couple of tabs, pressing the
   browser Back button would unexpectedly fall all the way out of the
   product onto the marketing landing page. Repeatable, observed,
   reported.

## Primary beneficiary personas

- **Customer Success / Sales engineering** — can finally send a
  "click here to see this view in your workspace" link.
- **Power users / admins** — can bookmark `/w/<id>?tab=calendar` for
  the views they look at every morning.
- **Anyone using the back button** (i.e. everyone) — the back button
  now works the way every browser-trained human expects.

## Marketing-ready claim

"Workspaces are first-class URLs. Send a link, bookmark a view, press
Back — all the things you take for granted on real web apps now work
in Effectime."

## Claim types enabled

- **Trust / professionalism** — back button working correctly is the
  bare minimum web users expect; getting it wrong felt amateurish.
- **Shareability** — workspace deep-links unlock async collaboration
  (drop a link in Slack instead of "go to the picker, find Acme, then
  click Calendar").
- **Multi-tenant maturity** — explicit workspace path makes it
  visually obvious which tenant you're in (the URL bar shows it).

  Fixing it removes a daily papercut.
- **Activation** — shareable URLs let admins onboard teammates by
  link, reducing the time-to-first-value.
