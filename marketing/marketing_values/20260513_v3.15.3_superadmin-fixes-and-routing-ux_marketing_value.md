# v3.15.3 — Superadmin fixes + routing-tree UX — Marketing Value

## Feature name and version

Superadmin Platform Control Plane: 3 hardening fixes + 2 new routing-tree
UX modes (v3.15.3).

## One-sentence user-language description

The Superadmin Platform Control Plane now actually shows workspaces and
platform metrics again, audit log loads with a real error if something
breaks (instead of `[object Object]`), and the feature-routing tree
gained a one-click "expand entire branch" button plus a flat-path
inventory view.

## Problem it replaces

A user testing the Superadmin Platform Control Plane reported three
visible failures in one screenshot:
- Overview cards all showed `—`
- Workspaces table rows had no name / locale / timezone
- Audit log threw `Az audit események betöltése nem sikerült: [object Object]`
plus two routing-tree pain points:
- No way to drill into a deep branch without clicking each level
- No "give me a flat copy-pasteable feature inventory" mode

## Primary beneficiary personas

- **Platform owner / operator** — can now use Platform Control Plane
  daily without bugs blocking the basic flow.
- **Customer Success / Sales engineering** — can copy a Flat view of
  the entire feature catalog into a customer-facing inventory.
- **UX / IA reviewer** — can expand a whole sub-tree with one click
  while reviewing menu hierarchy.

## Marketing-ready claim

"Three Superadmin regressions caught, fixed, and shipped within a single
release cycle. Platform Control Plane stays useful as the app grows
because every UI surface there is now wired to the same audit + routing
infrastructure the rest of the app relies on."

## Claim types enabled

- **Reliability** — concrete defect repair, visible to anyone using
  the Superadmin role.
- **Auditability** — the platform-wide audit log table now exists and
  is queryable, so future tier/addon changes have a permanent trail.
- **Operator efficiency** — one-click subtree expansion and flat view
  cut the time to assemble a "what's in the catalog" snapshot from
  minutes to seconds.
