# Visual Quality Checklist

## Purpose
Review checklist for all Effectime visual assets before publication. Use for: ad creatives, website images, social media graphics, case study visuals, infographics, and any branded visual output.

---

## Brand compliance checks

- [ ] Colors match approved brand palette (no ad-hoc colors introduced)
- [ ] Typography matches brand typeface guidelines (no random font choices)
- [ ] Logo used correctly: minimum clear space, correct color version for background
- [ ] No stretched, distorted, or poorly cropped images
- [ ] Icon style consistent with brand icon set (not mixing flat, outline, and 3D)

---

## Message clarity checks

- [ ] Primary message readable in 3 seconds (squint test)
- [ ] Headline and CTA visible at display size, not just at full resolution
- [ ] Text contrast meets accessibility standards (WCAG AA minimum)
- [ ] No text smaller than 14px in final render (12px for legal disclaimers only)
- [ ] Visual hierarchy is clear: headline → supporting message → CTA

---

## Content accuracy checks

- [ ] Product screenshots use realistic data (not "Test User 1" or obvious dummy data)
- [ ] Any claim shown in the visual is approved per terminology_and_positioning_guardrails.md
- [ ] No competitor logos or trademarks used without permission
- [ ] Statistics cited have a source if shown in infographic format
- [ ] Feature UI shown matches current product (not a deprecated screen)

---

## Channel-fit checks

- [ ] Dimensions are correct for the target platform and placement
- [ ] Image passes platform safety review (no misleading visuals, no prohibited content)
- [ ] Alt text written for accessibility (website, email, social)
- [ ] Mobile rendering checked (especially for email and website images)
- [ ] Animation or video: auto-play muted, captions available

---

## Style consistency checks

- [ ] Visual tone matches the content context (professional for ads, slightly warmer for founder content)
- [ ] Photography style: real workplace or clean abstract — not stock-photo-cliché (no group of smiling people pointing at a screen)
- [ ] No visual clutter: clean hierarchy, appropriate whitespace
- [ ] Color usage purposeful (not decorative overload)

---

## Final gate

All visual assets must pass this checklist before being scheduled or submitted to a platform. If any item is unclear, escalate to brand review before publishing.
